# Hi there! I'm Diogo 👋

I'm a DevOps Engineer at [RUAG](https://www.ruag.ch/en), Switzerland's Armed Forces technology partner.

Programming turned my world upside down when as a teen I decided to drop out of school and join **[42 Lausanne](https://42lausanne.ch)**, a private tuition-free coding institution where learning unfolds without teachers. Instead students are encouraged to be self-taught and learn through projects and practice. Since then I've kept pushing my limits through various projects and hackathons!
